# Запеканка / маффины из теста для панкейков

**Вариант для тех, кто не хочет стоять у плиты**
**Время:** ~5 мин подготовка + 25–30 мин духовка

---

## Тесто (то же самое что для панкейков)

- йогурт — 250 г (или коттедж)
- яйца — 2 шт
- мука — 150–160 г
- разрыхлитель — 1 ч.л.
- сахар / заменитель — по вкусу
- соль — щепотка
- ваниль или корица

---

## Вариант А — Большая запеканка

1. Форму 20–22 см смазать маслом
2. Вылить тесто (слой 2–3 см)
3. Сверху — кусочки яблок, банана, ягоды — что есть
4. 180°C, 25–30 минут
5. Проверить шпажкой — должна выходить сухой
6. Нарезать квадратиками — удобно есть руками (Талю тоже)

## Вариант Б — Мини-маффины

1. Разлить тесто по формочкам для маффинов (на ⅔)
2. Внутрь можно бросить ягоду, кусочек банана или яблока
3. 180°C, 20–25 минут
4. Хранятся 2–3 дня — удобно для перекусов

---

## Плюсы

- Не нужно стоять у плиты и переворачивать
- Залил — поставил — забыл
- Порционные маффины удобны ребёнку
